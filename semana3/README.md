El primer Programa del lunes se trato de realizar una conversion de la temperatura de celsius a fahrenheit y a kelvin. Lo primero que se hace es pedirle al usuario un numero a convertir, despues con ese numero se realiza las operaciones e imprime el resultado en las variables declaradas como K, F y C.
El segundo programa se trata de realizar unas operaciones indicadas por la doctora, para que estas puedan funcionar se añade la biblioteca math.h.
El tercer programa nos pide que transformemos coordenadas cartesianas a polares.Primero pedimos al usuario tres numeros para que transforme (x,y,z) a coordenadas polares (ro,theta,phi), se añade la biblioteca math.h para que las operaciones con funciones trigonometricas puedan activarse. Por ultimo de este programa imprimimos el resultado de las operciones en las variables ro, theta y phi.
El primer ejercicio de miercoles trata de usar la sentencia switch la cual permite seleccionar las acciones a realizar de acuerdo al valor que tome una variable.Mi programa fue de pedirle un numero al usuario del 1 al 3 para despues imprimirlo en  la pantalla. Despues de acuerdo al numero de casos que tiene la sentencia switch imprimia el numero que el ususario escribio (1-3). Si el usuario escribe un numero diferente a 1 a 3 se imprime que es incorrecto.
El segundo y tercer programa del miercoles son muy similares ya que se tratan de dividir un numero y decir si es par o impar, en este caso con la estructura condicional if else en la cual se agrega una condicion, en este programa divide el numero inscrito por el usuario y si el cociente es 0 entonces el numero es par, si no el numero sera impar. el tercero lo unico que cambia es que si el numero es par se le suma 1, esto se resolvio con añadir una variable (a2).
El 4to y 5to programa del miercoles son del mismo tema (condicionales if-else). en el cuarto la condicional tenia que comparar dos numeros ingresados por el usuario (si son iguales, si son mayor o menor que el otro numero).El 5to programa imprime la letra vocal que tecleo el usuario y dice si es mayuscula o minuscul, a demas si no se tecleo una vocal el programa marca que es una consonante.
